A modern, beautifully designed chatbot with AI-powered responses powered by Groq's Llama-3.3-70B model.

## ✨ Features

- **🎨 Modern Glassmorphism UI** - Stunning dark theme with animated backgrounds and smooth transitions
- **🤖 AI-Powered Responses** - Intelligent conversations using Groq's Llama-3.3-70B model
- **💬 Conversation History** - Maintains context across multiple messages
- **⚡ Quick Actions** - Pre-built buttons for common interactions
- **🔒 API Key Management** - Secure local storage of your API credentials
- **📱 Responsive Design** - Works perfectly on desktop and mobile
- **✨ Beautiful Animations** - Smooth entrance, typing indicators, and micro-interactions

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A Groq API key (free at [console.groq.com](https://console.groq.com/keys))

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/pratikbug24/chat_bot.git
   cd chat_bot
   ```

2. **Get your Groq API key**
   - Visit [console.groq.com/keys](https://console.groq.com/keys)
   - Create a free account (if you don't have one)
   - Generate a new API key

3. **Configure the chatbot**
   - Open `index.html` in your browser
   - Click the ⚙️ **settings icon** in the header
   - Paste your API key and click **Connect to Groq AI**

4. **Start chatting!**

## 🎯 Usage Tips

- **Quick Actions**: Use the pre-built buttons for common queries
  - Say Hi - Start a friendly conversation
  - C Language - Get programming insights
  - Ask Question - Get help with any topic

- **Keyboard Shortcuts**:
  - Press **Enter** to send a message
  - Use **Shift + Enter** for new lines

- **API Status**:
  - 🟢 Green dot = Connected to Groq AI
  - ⚪ Gray dot = Using fallback responses
